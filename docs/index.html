核心功能

🔍

多平台即時監控

同時監控蝦皮（Shopee）與露天拍賣（Ruten）兩大電商平台。 以 Playwright 驅動真實瀏覽器，有效繞過反爬蟲限制，精準抓取最新上架商品。

🎮

Discord 即時通知

發現新商品時立即透過 Discord Webhook 推送 Embed 通知，包含商品圖片、名稱、價格與連結。 支援 @ 提及功能，不錯過任何商品。

⚙️

彈性關鍵字管理

透過 Next.js 後台管理監控關鍵字，支援多關鍵字、選擇平台、設定最低／最高價格範圍。 已通知過的商品自動去重，不重複打擾。

快速上手

1

安裝依賴

安裝所有 Python 套件（需要 Python 3.11+）：

pip install -r requirements.txt
2

安裝 Playwright 瀏覽器

下載 Chromium 及其系統依賴：

playwright install --with-deps chromium
3

設定環境變數

複製範本並填入你的設定：

cp .env.example .env
# 編輯 .env，填入 WORKER_SECRET 與 NEXT_PUBLIC_API_URL
4

啟動 Worker

啟動後每 5 分鐘自動掃描一次：

python main.py

環境變數參考

變數名稱 必填 預設值 說明
WORKER_SECRET 必填 — 與 Next.js API 共享的驗證密鑰
NEXT_PUBLIC_API_URL 必填 — Next.js API base URL（例：https://xxx.vercel.app）
CHECK_INTERVAL 選填 300 掃描間隔秒數（預設 5 分鐘）

🐳 Docker 部署

使用官方 Playwright Python 映像，無需手動安裝瀏覽器：

# 建置映像
docker build -t shop-watcher .

# 啟動 Worker（替換為你的實際值）
docker run -d \
  -e WORKER_SECRET=your_secret_here \
  -e NEXT_PUBLIC_API_URL=https://your-app.vercel.app \
  --name shop-watcher \
  shop-watcher

映像基於 mcr.microsoft.com/playwright/python:v1.44.0-focal，已內建 Chromium。

🚀 Fly.io 雲端部署

Worker 為無狀態設計，不需要 persistent volume，直接部署即可： 

# 初始化（只需執行一次）
fly launch

# 設定 Secrets（不寫入程式碼）
fly secrets set \
  WORKER_SECRET=your_secret_here \
  NEXT_PUBLIC_API_URL=https://your-app.vercel.app

# 部署
fly deploy

Fly.io 免費方案包含足夠的 compute hours，適合 Worker 長期運行。
